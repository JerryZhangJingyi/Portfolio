JK Bank - a Database Project

This project can help a bank to collect, store and observe its customers' information such as name, address, card number, deposit amount and so on. JK Bank's interface was written in html and php, and the database was constructed by using sql. This was my first project that is related to database, and I chose this project because it helped me to gain understandings about database managements. It was also helpful for me to learn how to build application interface by using html and php.

4 Pong - a Programing Language Project

4 Pong is a mini game which allows 2 teams (4 users) to play a pong game together, the game's rule adjusts as each team earns more scores. 4 Pong was written in html and Elm. This project helped me to develop my problem solving skills since Elm was a relatively new language and it requires some learnings to build an application by using Elm.

https://github.com/MerrittRuthrauff/4pong

Film Jamz - a Datamining Project

Film Jamz shows the analysed results of movie industry in various graphs. The objective for Film Jamz was to find out what entity (director, actor, genre, rating etcetera) affects movies' gross income the most. Decision Tree Regressor, Random Forest Regressor, Gradient Boosting Regressor, KNeighbors Regressor and AdaBoost Regressor were used as algoritms and the graphs were written in Python. This project helped me to gain more knowledge on collecting, sorting, pre-processing and analysing data.

https://github.com/aidanpirish/filmjamz

PiTando - a Creative Eletronic Project

PiTendo allows users to run old fashion games from various consoles and play them by using XBox controllers. It was built by using Raspberry Pi 3 kits, Mausberry circuit and it runs on RetroPi OS. This project focused more on the hardware aspect and it helped me to learn more about how programs interact with computer hardware.

Space Speller - a Software Engineering Project

Space Speller was a game created for elementary students to learn spelling, it also records each student's high scores in database for teachers to observe. The game was written in Javascript, JQuery and it uses Phaser game engine. The web app runs from a Django server. I chose this project in order to challenge myself to build a more complex project. This project helped me to learn how web app interacts with server. It also helped me to acquire exprience to work in a group and gain more understandings for software design, because there were 4 contributors and I specifically worked on Django along with the database.

https://github.com/Rosewired/JBEXsoftware
